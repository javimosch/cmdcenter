June 2026

Command Center v1.0.0

🎨

Minimalist UI Redesign

Complete UI overhaul with warm monochrome palette, Inter and JetBrains Mono fonts, and premium editorial aesthetic. Removed gradients, bright colors, and heavy shadows for a cleaner, more professional appearance.

🔗

Link Type Support

Added support for link-type commands that open URLs instead of executing shell commands. Commands can now be categorized as either shell commands or web links with unified management interface.

📂

Category Grouping

Implemented category-based organization for commands and links. Added category tabs for filtering and better organization of the command dashboard.

⚙️

Environment Variables

Added support for environment variables in command execution. Commands can now specify KEY=VALUE pairs that are set before execution for better configuration management.

🐛

Modal Positioning Fix

Fixed modal positioning to be properly centered in viewport regardless of page scroll. Added body scroll lock when modals are open to prevent background scrolling.

🐳

Docker Deployment

Added Docker deployment support with compose files for both development and production. Includes Traefik integration for automatic SSL and domain routing.
